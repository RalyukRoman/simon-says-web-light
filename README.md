# Simon Says (Light Version)

This is a lightweight version of the Simon Says game built with Next.js. Players follow sequences of colors to test memory and reaction skills.

## Launch
1. Install dependencies: npm install or yarn install
2. Start the development server: npm run dev or yarn dev
3. Open http://localhost:3000 in your browser to play Simon Says
